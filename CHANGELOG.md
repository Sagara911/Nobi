# 更新日志

每个版本简短记一句改了什么。**发版时，本文件里对应版本(`## v版本号`)的条目会被 CI 自动取出，
作为 GitHub Release 说明 + 应用内「发现新版本」弹窗里显示的更新内容**（见 `.github/workflows/release.yml`）。
所以发版前，先在最上面加一条本版条目。

## v0.2.20
- 浏览窗彻底隐身：按 Alt+3 调出标题栏也不再露到任务栏/Alt+Tab，全程除老板键外任何方式都看不见（和便签一致）

## v0.2.19
- 便签改为置顶悬浮（和浏览窗一样浮在其它窗口最上层）

## v0.2.18
- 修复便签的「隐藏 / 最大化」和浏览窗 Alt+3 的窗口按钮只闪一下就消失（之前窗口一获焦就被打回隐身样式）；现在隐身只在首次打开时设一次，不再打断后续操作

## v0.2.17
- 聊天里能玩飞行棋了：右上角 🎲 开局，朋友点加入即可对战（2–4 人，掷 6 起飞、撞子回家、四子到家获胜）
- （本版飞行棋为基础玩法，暂不含「飞」和捷径跳跃，后续再加）

## v0.2.16
- UNO：修复打完一局后卡在结算页、开不了下一把——结算页给所有人加「退出」，房主可「再来一局」；出牌后即时显示「出牌中…」反馈，按钮/卡牌有按下感
- 便签：透明度首次打开就生效了（之前要点一下窗口才变透明）
- 便签：标题栏补上「隐藏 / 最大化」按钮（隐藏后用老板键 Alt+C 或托盘唤回）
- 浏览窗：按 Alt+3 显示标题栏时带上完整的最小化 / 最大化 / 关闭（此时会临时出现在任务栏），收起标题栏恢复隐身

## v0.2.14
- 聊天：可改自己的气泡颜色（头像面板里选，8 个预设 + 自定义取色；字色自动适配深浅）
- 聊天里能玩 UNO：右上角 🎴 开局，朋友点加入即可对战（2–4 人）
- 画板：从桌面拖图到画板上现在直接落在画板上（临时、不入库）；想存进素材库可在画板里右键图片「保存到素材库」

## v0.2.0
- 内置翻译大升级：划词翻译支持多语种；英文查词走离线词典（音标 + 释义，约 3.7 万常用词随包，断网也能查）
- 整句可离线翻译：内置神经翻译模型（中英互译），有本地大模型时优先用、没有或断网也能翻；模型首次使用时按需下载
- 技术词（python3、ai_config 等）不再被乱翻成「蟒蛇」；桌面划词 / 应用内 / 浏览器扩展三处统一显示音标 + 释义
- 新增「和朋友聊天」：默认走云端，支持并排开多个群、桌面拖图直接发送、Alt+C 老板键一键隐藏所有聊天窗

## v0.1.15
- 素材库侧栏精简：去掉配色分类、标签默认只列常用的（冷门收进「全部标签」）、智能合集收成小按钮
- 网格新增 图片 / 视频 / 音频 快捷筛选；所有筛选都支持「再点一下取消、回到全部」
- 移除 3D 预览（本机环境无法稳定渲染），安装包瘦身

## v0.1.14
- 看球小窗快捷键可自定义（逐个改键 + 冲突检测 + 恢复默认）
- 老板键：藏窗时自动暂停视频、显示时自动续播
- 弹窗按钮样式统一

## v0.1.13
- 看球小窗重做：以「直开网页」为主，登录 / 全屏 / 站内跳转都正常
- 一排快捷键（透明度 / 缩放 / 穿透 / 换台搜索 / 截图入库 / 贴角 / 老板键）
- 记住窗口大小位置、透明度、搜索引擎
