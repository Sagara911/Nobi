# 更新日志

每个版本简短记一句改了什么。**发版时，本文件里对应版本(`## v版本号`)的条目会被 CI 自动取出，
作为 GitHub Release 说明 + 应用内「发现新版本」弹窗里显示的更新内容**（见 `.github/workflows/release.yml`）。
所以发版前，先在最上面加一条本版条目。
